# Cát Phương Nam — Portfolio Starter

## Deploy in about 5 minutes

1. Create a GitHub repository named exactly `phuongnamcatier.github.io`.
2. Upload all files from this package into the repository root.
3. Open `Settings → Pages`.
4. Under `Build and deployment`, choose `Deploy from a branch`.
5. Select branch `main` and folder `/ (root)`, then Save.
6. Visit `https://phuongnamcatier.github.io`.

GitHub may take several minutes to publish the first version.

## Edit these first

- Email and LinkedIn: `index.html`
- Homepage content: `index.html`
- Colors/layout: `styles.css`
- Case studies: files inside `projects/`
- Project images: `assets/projects/`
- CV: `assets/cv/CV-Cat-Phuong-Nam.docx`

## Git command option

```bash
git init
git add .
git commit -m "Launch portfolio"
git branch -M main
git remote add origin https://github.com/phuongnamcatier/phuongnamcatier.github.io.git
git push -u origin main
```

No framework or build step is required.
